# 社团活动记录

记录我们社团的各种活动和事件。

## 2024年活动

### 定期活动

#### 每周学习会
- **时间**：每周六下午 2:00-5:00
- **地点**：图书馆研讨室
- **内容**：
  - 回顾本周课程内容
  - 讨论Problem Set
  - 互相答疑

#### 代码审查会
- **时间**：每周日晚上 7:00-9:00
- **地点**：线上（Zoom/腾讯会议）
- **内容**：
  - 分享自己的代码实现
  - 学习他人的优秀方案
  - 提升代码质量

### 特别活动

#### 黑客马拉松
- **日期**：待定
- **主题**：使用所学知识开发实际项目
- **奖励**：优秀项目将获得奖品

#### 讲座和工作坊
- 邀请业界专家分享经验
- 深入学习特定主题
- 职业发展指导

# 学习小组

为了更好地学习和互助，我们组建了不同的学习小组。

## 小组分类

### 按进度分组

#### 入门组
- **适合对象**：刚开始学习CS50的成员
- **主要内容**：Week 0-2
- **学习节奏**：较慢，注重基础理解

#### 进阶组
- **适合对象**：有一定编程基础的成员
- **主要内容**：Week 3-5
- **学习节奏**：正常进度

#### 冲刺组
- **适合对象**：想快速完成课程的成员
- **主要内容**：所有周次
- **学习节奏**：较快，需要投入更多时间

### 按兴趣分组

#### Web开发组
关注Web技术，特别是Week 8（HTML, CSS, JavaScript）以后的内容。

#### 算法研究组
深入研究算法和数据结构，参加算法竞赛。

#### 系统编程组
对底层编程、操作系统感兴趣的成员。

## 加入小组

1. 确定自己的学习目标和节奏
2. 选择合适的小组
3. 联系小组组长
4. 参与小组活动

## 小组活动

- 每周固定时间讨论
- 一起解决Problem Set
- 分享学习资源
- 互相答疑

## 成为组长

如果你想创建新的学习小组：
1. 确定小组主题和目标
2. 制定学习计划
3. 向社团管理员申请
4. 招募组员

## 小组联系方式

具体联系方式请查看社团群公告或联系管理员。

## 学习建议

- 保持规律的学习习惯
- 积极参与讨论
- 不要害怕提问
- 乐于帮助他人
- 坚持完成Problem Set

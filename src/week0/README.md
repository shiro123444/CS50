# Week 0 - Scratch

## 本周主题

本周我们将学习编程的基础概念，使用Scratch这个可视化编程语言来创建我们的第一个程序。

## 主要内容

### 计算思维

- 问题分解
- 模式识别
- 抽象思维
- 算法设计

### Scratch基础

- **事件**：程序如何开始执行
- **循环**：重复执行代码
- **条件**：根据不同情况做出决策
- **变量**：存储和使用数据
- **函数**：组织和重用代码

尝试用Scratch创建：
- 一个简单的动画
- 一个互动小游戏
- 一个问答程序

## 小组讨论

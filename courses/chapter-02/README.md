# Week 1: C 语言基础

欢迎来到 Week 1！本周我们将从 Scratch 的可视化编程过渡到 C 语言的文本编程。C 语言是一门强大的底层编程语言，学习它将帮助你理解计算机的工作原理。

## 学习目标

通过本周学习，你将能够：

- 理解 C 语言的基本语法
- 掌握变量、数据类型和运算符
- 使用条件语句和循环结构
- 编写和编译简单的 C 程序
- 理解命令行界面的使用

## 章节内容

本章包含以下课程：

1. [从 Scratch 到 C](lesson-01.md) - 理解可视化编程到文本编程的转换
2. [C 语言基础语法](lesson-02.md) - 学习 C 语言的核心语法和概念

## 预计学习时间

约 8-12 小时（包括观看讲座和完成作业）

## 前置知识

- 完成 Week 0 的 Scratch 学习
- 理解基本的编程概念（变量、循环、条件）

## 本周作业

Problem Set 1 包含多个编程练习：
- **Hello** - 编写第一个 C 程序
- **Mario** - 使用循环打印金字塔图案
- **Cash/Credit** - 实现信用卡验证算法

## 重要概念

- 编译器（Compiler）
- 数据类型（int, float, char, string）
- 条件判断（if, else if, else）
- 循环（while, for）
- 函数（Functions）

准备好迎接真正的编程挑战了吗？让我们开始吧！

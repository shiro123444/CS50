# Week 0: Scratch - 计算思维入门

欢迎来到 CS50x 2025 的第一周！本周我们将通过 Scratch 这个可视化编程语言来学习计算机科学的基础概念。

## 学习目标

通过本周学习，你将能够：

- 理解计算思维的核心概念
- 掌握算法、循环、条件判断等编程基础
- 使用 Scratch 创建交互式项目
- 理解函数、变量和事件的概念

## 章节内容

本章包含以下课程：

1. [CS50 课程介绍](lesson-01.md) - 了解 CS50 课程内容和学习方法
2. [Scratch 编程基础](lesson-02.md) - 学习使用 Scratch 进行可视化编程

## 预计学习时间

约 6-8 小时（包括观看讲座和完成作业）

## 前置知识

- 无需编程经验
- 对计算机基本操作有了解

## 本周作业

创建一个 Scratch 项目，要求：
- 至少包含两个精灵（sprites）
- 使用至少三个脚本
- 包含条件判断和循环
- 具有交互性

让我们开始这段激动人心的计算机科学之旅吧！

# 学术论文模板与写作规范

## 论文写作提示词模板

### 核心要求

**写作风格**:
- 使用第三人称客观叙述
- 避免口语化表达
- 使用学术规范的专业术语
- 保持逻辑严谨、论证充分

**格式规范**:
- 遵循学术论文标准结构
- 使用标准的章节编号系统
- 包含完整的参考文献引用
- 提供详细的数据和实验结果

---

## 标准学术论文结构

### 1. 标题页

```
论文标题（中英文）
作者信息
单位信息
摘要（中英文，200-300字）
关键词（3-5个）
```

### 2. 正文结构

#### 2.1 引言 (Introduction)
- 研究背景
- 问题陈述
- 研究意义
- 论文组织结构

#### 2.2 相关工作 (Related Work)
- 文献综述
- 现有方法分析
- 本研究的创新点

#### 2.3 方法论 (Methodology)
- 系统架构
- 核心算法
- 技术实现
- 数学模型

#### 2.4 实验设计 (Experimental Design)
- 实验环境
- 数据集描述
- 评估指标
- 实验流程

#### 2.5 结果与分析 (Results and Analysis)
- 实验结果
- 性能对比
- 结果分析
- 讨论

#### 2.6 结论 (Conclusion)
- 主要贡献
- 局限性
- 未来工作

#### 2.7 参考文献 (References)
- 标准格式引用
- 按字母顺序或引用顺序排列

## 具体案例写作规范

### 案例论文标题格式

**中文标题**：基于[技术/方法]的[应用领域][系统/算法]设计与实现  
**英文标题**：Design and Implementation of [System/Algorithm] Based on [Technology/Method] for [Application Domain]

### 摘要写作要求

**结构**：
1. 研究背景（1-2句）
2. 问题陈述（1句）
3. 提出方法（2-3句）
4. 主要结果（1-2句）
5. 研究意义（1句）

**示例**：
```
摘要：随着人工智能技术的快速发展，智能体系统在多个领域展现出广泛的应用前景。
然而，现有智能体框架在工具集成和工作流编排方面存在标准化程度低、可扩展性差等问题。
本文提出了一种基于Model Context Protocol的智能体工作流系统，
通过设计顺序执行器架构和动态工具生命周期管理机制，
实现了标准化的工具接成和灵活的工作流编排。
实验结果表明，该系统在内容生产任务中表现出良好的性能和可扩展性，
为智能体系统的实际应用提供了有价值的参考。

关键词：智能体系统；Model Context Protocol；工作流编排；工具集成
```

## 数学公式规范

### 公式编号

独立公式使用编号：
```
设X为输入数据矩阵，Y为参考信号矩阵，则典型相关分析的目标函数为：

    ρ = max corr(X·a, Y·b)     (1)

其中a和b分别为X和Y的典型相关向量。
```

### 符号说明

在首次使用数学符号时提供明确定义：
```
令：
- X ∈ ℝ^(n×p): 脑电信号矩阵，n为采样点数，p为通道数
- Y ∈ ℝ^(n×q): 参考信号矩阵，q为谐波数
- ρ ∈ [0,1]: 典型相关系数
```

## 教学案例特殊要求

### 教学目标描述

**格式**：
```
本案例的教学目标包括：
(1) 知识目标：使学生掌握...的基本原理和实现方法；
(2) 能力目标：培养学生...的实践能力；
(3) 素质目标：提升学生的...思维和...意识。
```

### 教学内容组织

**格式**：
```
教学内容分为以下几个模块：

模块一：理论基础（2课时）
    内容：...的基本概念、数学模型、算法原理
    要求：学生能够理解...，掌握...

模块二：系统设计（2课时）
    内容：架构设计、模块划分、接口定义
    要求：学生能够独立完成...的设计

模块三：实践开发（2课时）
    内容：代码实现、测试验证、性能优化
    要求：学生能够实现...功能
```

# CS50 课程介绍

## 什么是 CS50？

CS50 是哈佛大学的计算机科学导论课程，也是全球最受欢迎的在线公开课之一。这门课程由 David J. Malan 教授主讲，以其生动有趣的教学方式和严谨的课程设计而闻名。

![CS50 Logo](../../resources/images/cs50-logo.png)

## 课程特色

### 1. 从零开始

- 无需任何编程经验
- 循序渐进的学习路径
- 注重计算思维的培养

### 2. 实践导向

CS50 强调动手实践：

- 每周都有编程作业（Problem Sets）
- 期末有个人项目（Final Project）
- 通过实际项目学习编程

### 3. 多语言学习

```
课程涵盖的编程语言：
- Scratch（可视化编程）
- C（底层编程）
- Python（高级编程）
- SQL（数据库）
- JavaScript（Web 开发）
- HTML/CSS（网页设计）
```

## CS50x 2025 课程大纲

| 周次 | 主题 | 编程语言 | 核心概念 |
|------|------|----------|----------|
| Week 0 | Scratch | Scratch | 计算思维、算法 |
| Week 1 | C | C | 变量、数据类型、条件 |
| Week 2 | Arrays | C | 数组、字符串、命令行 |
| Week 3 | Algorithms | C | 搜索、排序、算法复杂度 |
| Week 4 | Memory | C | 指针、内存管理 |
| Week 5 | Data Structures | C | 链表、树、哈希表 |
| Week 6 | Python | Python | Python 基础 |
| Week 7 | SQL | SQL | 数据库、关系型数据 |
| Week 8 | HTML, CSS, JavaScript | Web | 前端开发 |
| Week 9 | Flask | Python | Web 应用开发 |
| Week 10 | Final Project | 自选 | 综合项目 |

## 学习资源

CS50 提供丰富的学习资源：

```
可用资源：
├── 视频讲座（Lectures）
├── 课堂笔记（Notes）
├── 编程作业（Problem Sets）
├── 实验练习（Labs）
├── 在线 IDE（CS50 IDE）
└── 讨论社区（Ed Discussion）
```

## 如何学习 CS50

### 学习建议

1. **观看讲座视频**
   - 完整观看每周的讲座
   - 做好笔记，理解核心概念
   - 可以调整播放速度

2. **完成实验和作业**
   - 先完成 Lab（实验）热身
   - 独立完成 Problem Set（作业）
   - 遵守学术诚信政策

3. **参与社区讨论**
   - 在 Ed Discussion 提问
   - 帮助其他学习者
   - 分享学习心得

4. **保持学习节奏**
   - 每周投入 10-20 小时
   - 不要拖延作业
   - 坚持完成所有内容

## 学术诚信

CS50 非常重视学术诚信：

- ✅ 可以与他人讨论问题思路
- ✅ 可以查阅在线资源和文档
- ❌ 不能直接复制他人代码
- ❌ 不能使用 AI 工具完成作业
- ❌ 不能查看或分享作业答案

## 评分标准

CS50 的评分维度：

| 维度 | 说明 | 权重 |
|------|------|------|
| Correctness | 代码正确性 | 50% |
| Design | 代码设计质量 | 25% |
| Style | 代码风格规范 | 25% |

## 完成课程后你将获得

- ✅ 扎实的编程基础
- ✅ 计算思维能力
- ✅ 多种编程语言经验
- ✅ 完整的项目作品集
- ✅ CS50 官方证书（可选付费）

## 课程内容

### Week 0: Scratch - 计算思维入门

通过 Scratch 可视化编程语言学习计算机科学的基础概念，包括算法、循环、条件判断等。

- [CS50 课程介绍](/courses/chapter-01/lesson-01) - 了解 CS50 课程内容和学习方法
- [Scratch 编程基础](/courses/chapter-01/lesson-02) - 学习使用 Scratch 进行可视化编程

### Week 1: C 语言基础

从 Scratch 过渡到 C 语言，学习文本编程的基础语法和核心概念。

- [从 Scratch 到 C](/courses/chapter-02/lesson-01) - 理解可视化编程到文本编程的转换
- [C 语言基础语法](/courses/chapter-02/lesson-02) - 学习 C 语言的核心语法和概念

## 关于 CS50

CS50 是哈佛大学的计算机科学导论课程，也是全球最受欢迎的在线公开课之一。这门课程由 David J. Malan 教授主讲，以其生动有趣的教学方式和严谨的课程设计而闻名。

### 课程特色

- **从零开始** - 无需任何编程经验
- **实践导向** - 每周都有编程作业和项目
- **多语言学习** - 涵盖 Scratch、C、Python、SQL、JavaScript 等
- **全球社区** - 数百万学习者共同学习

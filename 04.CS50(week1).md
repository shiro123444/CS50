### 2. 命令行编辑快捷键 (光标与控制)
**这是提升效率的关键**。当你输错命令或想找回之前的命令时，不要疯狂按退格键，用这些组合键：

*   **Tab**: **自动补全** (必用！输入文件名或命令的前几个字母，按 Tab 自动补全)
*   **↑ / ↓** (上下箭头): 翻阅**上一条/下一条**历史命令
*   **Ctrl + C**: **强制终止**当前正在运行的程序 (例如退不出的服务器)
*   **Ctrl + L**: **清屏** (等于 `clear` 命令)

| 快捷键 | 作用 | 记忆技巧 |

| `Ctrl + E` | 光标跳到行**尾** | **E**nd |
| `Ctrl + U` | 删除光标**左侧**所有内容 | 清空当前行输入 |
| `Ctrl + K` | 删除光标**右侧**所有内容 | Kill |
| `Ctrl + W` | 删除光标左侧的一个**单词** | Word |
| `Ctrl + R` | **搜索历史命令** | Recall (输入关键字查找之前输过的长命令) |

### 4. 💡 进阶小贴士 (Aliases)
如果你觉得 `git status` 或 `npm run dev` 打字太累，可以在你的 Shell 配置文件（`.bashrc` 或 `.zshrc`）里设置 **别名 (Alias)**：

```bash
# 例子：
alias c="clear"
alias gs="git status"
alias ga="git add ."
alias gc="git commit -m"
alias nr="npm run"
```
这样你只需要输入 `gs` 就能查看 git 状态了。

6.**==基础语法==**

这份 C 语言基础语法总结旨在作为一份**速查表 (Cheat Sheet)**，涵盖了核心概念。你可以把它当作复习资料，或者在写代码卡壳时查阅。

### 1. 程序基本骨架
所有的 C 程序都从 `main` 函数开始执行。

```c
#include <stdio.h> // 引入标准输入输出库 (Standard Input/Output)

int main() {
    // 你的代码写在这里
    printf("Hello, World!\n");
    
    return 0; // 返回 0 表示程序正常结束
}
```

### 2. 数据类型与变量
在 C 语言中，使用变量前**必须先声明**。

| 类型 | 描述 | 占用字节 (通常) | 格式占位符 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| **int** | 整数 | 4 | `%d` | `int age = 18;` |
| **float** | 单精度浮点数 | 4 | `%f` | `float pi = 3.14;` |
| **double** | 双精度浮点数 | 8 | `%lf` | `double bigPi = 3.141592;` |
| **char** | 单个字符 | 1 | `%c` | `char grade = 'A';` |
| **char[]**| 字符串 (字符数组) | - | `%s` | `char name[] = "C";` |

*   **const**: 定义常量（不可修改）。
    ```c
    const float PI = 3.14; // 此后 PI 不能被重新赋值
    ```

### 3. 输入与输出
使用 `<stdio.h>` 库。

*   **输出 (printf)**:
    ```c
    int money = 100;
    // \n 表示换行，%.2f 表示保留两位小数
    printf("I have %d dollars and %.2f cents.\n", money, 50.0);
    ```

*   **输入 (scanf)**: **(注意：除了字符串，变量前都要加 `&` 取地址)**
    ```c
    int age;
    printf("请输入年龄: ");
    scanf("%d", &age); // 这里的 & 很重要！
    ```

### 4. 运算符 (Operators)
这是逻辑的核心，注意区分赋值和比较。

#### A. 算术运算符
| 符号 | 含义 | 示例 | 注意事项 |
| :--- | :--- | :--- | :--- |
| `+` `-` `*` | 加、减、乘 | `a * b` | |
| `/` | 除法 | `5 / 2` 结果是 `2` | 整数相除只保留整数部分！要得到 `2.5` 需写成 `5.0 / 2` |
| `%` | **取模 (取余数)** | `5 % 2` 结果是 `1` | 只能用于整数 |
| `++` | 自增 | `i++` (先用后加) / `++i` (先加后用) | 等同于 `i = i + 1` |
| `--` | 自减 | `i--` | 等同于 `i = i - 1` |

#### B. 关系运算符 (比较)
结果为 **1 (真)** 或 **0 (假)**。
| 符号 | 含义 | 常见坑点 |
| :--- | :--- | :--- |
| `==` | **等于** | **千万别写成 `=` (赋值)** |
| `!=` | 不等于 | |
| `>` `<` | 大于、小于 | |
| `>=` `<=` | 大于等于、小于等于 | |

#### C. 逻辑运算符 (组合条件)
| 符号 | 含义 | 示例 | 说明 |
| :--- | :--- | :--- | :--- |
| `&&` | **并且 (AND)** | `(a > 0) && (a < 10)` | 两边都为真，结果才为真 |
| `\|\|` | **或者 (OR)** | `(a == 1) || (a == 2)` | 只要有一边为真，结果就为真 |
| `!` | **非 (NOT)** | `!(a == 0)` | 取反，真变假，假变真 |

#### D. 赋值运算符
*   `=` : 赋值 (将右边的值给左边)
*   `+=` : `a += 5` 等同于 `a = a + 5`
*   `-=` : `a -= 5` 等同于 `a = a - 5`

### 5. 流程控制

#### A. 条件判断 (if...else)
```c
if (score >= 90) {
    printf("优秀");
} else if (score >= 60) {
    printf("及格");
} else {
    printf("挂科");
}
```

#### B. 选择结构 (switch)
适用于针对同一个变量的多个固定值判断。
```c
switch (grade) {
    case 'A':
        printf("Great!");
        break; // 别忘了 break，否则会继续执行下一个 case
    case 'B':
        printf("Good");
        break;
    default:
        printf("Unknown");
}
```

#### C. 循环 (Loop)
*   **for 循环** (最常用，已知次数)：
    ```c
    // (初始化; 条件; 迭代)
    for (int i = 0; i < 5; i++) {
        printf("%d ", i);
    }
    ```
*   **while 循环** (条件满足时一直执行)：
    ```c
    while (battery > 0) {
        printf("Playing game...");
        battery--;
    }
    ```

### 6. 数组 (Array)
存储一组相同类型的数据。**下标从 0 开始**。

```c
int numbers[5] = {10, 20, 30, 40, 50};

// 访问第三个元素
printf("%d", numbers[2]); // 输出 30

// 修改元素
numbers[0] = 99;
```

### 7. 函数 (Function)
将代码模块化。
```c
// 函数声明：返回类型 函数名(参数类型 参数名)
int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 5); // 调用函数
    return 0;
}
```
*   **void**: 如果函数不返回任何值，返回类型写 `void`。

### 8. 指针入门 (Pointer) - ⚠️ 难点
指针就是**存储内存地址**的变量。

| 符号 | 含义 | 助记 |
| :--- | :--- | :--- |
| `&` | **取地址符** | 获取变量在内存中的“门牌号” |
| `*` | **解引用符** | 根据“门牌号”取出里面的数据 |

```c
int a = 10;
int *p = &a; // p 是一个指针，存了 a 的地址

printf("%p", p);  // 输出地址 (类似 0x7ffe...)
printf("%d", *p); // 输出 10 (访问 p 指向的地址里的值)

*p = 20; // 通过指针修改 a 的值
// 此时 a 也变成了 20
```

### 💡 新手避坑指南
1.  **分号**：每句代码结束记得加 `;`。
2.  **等于号**：比较相等是 `==`，赋值是 `=`。
3.  **整数除法**：`1 / 2` 结果是 `0`。如果想要小数，必须写成 `1.0 / 2`。
4.  **数组越界**：如果你定义了 `int a[5]`，那么有效的下标是 `0` 到 `4`，访问 `a[5]` 会报错或出现乱码。
5.  **scanf 里的 &**：读取 `int`, `float`, `char` 必须加 `&`，读取字符串 (`char` 数组) 不需要加 `&`。

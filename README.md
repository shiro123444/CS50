# 社团课程学习平台

欢迎来到我们的在线学习平台！这是一个基于 GitBook 构建的课程学习系统，旨在为社团成员提供结构化的学习内容和互动交流空间。

## 平台特色

- **📚 结构化课程内容**: 清晰的章节组织，循序渐进的学习路径
- **🔍 强大的搜索功能**: 快速查找所需的学习资料
- **💬 互动讨论区**: 在每个课程页面下与其他成员交流学习心得
- **❓ 问题收集系统**: 遇到问题？随时提交 Issue 获得帮助
- **📱 响应式设计**: 支持手机、平板和电脑等多种设备访问

## 使用指南

### 浏览课程

使用左侧导航栏浏览所有可用的课程章节。点击章节标题展开查看具体的课程内容。

### 搜索内容

使用页面顶部的搜索框快速查找相关内容。支持关键词搜索，帮助您快速定位所需信息。

### 参与讨论

每个课程页面底部都有基于 Giscus 的评论区，您可以：
- 💬 分享学习心得和经验
- ❓ 提出疑问并获得解答
- 🤝 与其他成员交流互动
- 👍 为有帮助的评论点赞

> 💡 提示：参与讨论需要 GitHub 账号。评论基于 GitHub Discussions，安全可靠。

### 提交问题

如果您在学习过程中遇到问题，可以通过以下方式提交：

1. 点击课程页面底部的"提交问题"链接
2. 选择合适的问题类型（学习问题、Bug 报告、功能建议）
3. 填写问题详情并提交
4. 等待管理员或其他成员的回复

📖 **详细指南**: 查看 [Issue 提交指南](.github/ISSUE_GUIDE.md) 了解如何有效地提交和管理问题

**快速链接**:
- [📝 提交学习问题](https://github.com/YOUR_USERNAME/YOUR_REPO/issues/new?template=question.md)
- [🐛 报告错误](https://github.com/YOUR_USERNAME/YOUR_REPO/issues/new?template=bug.md)
- [💡 提出建议](https://github.com/YOUR_USERNAME/YOUR_REPO/issues/new?template=enhancement.md)
- [📋 查看所有 Issues](https://github.com/YOUR_USERNAME/YOUR_REPO/issues)

### 移动端访问

本平台完全支持移动设备访问，您可以随时随地学习课程内容。

## 内容更新

课程内容会定期更新和完善。所有更新都会通过 Git 版本控制系统进行管理，确保内容的准确性和可追溯性。

## 技术支持

如果您在使用平台过程中遇到任何技术问题，请：
- 查看相关课程的使用说明
- 在讨论区寻求帮助
- 提交 GitHub Issue 报告问题

## 开始学习

准备好了吗？点击左侧导航栏开始您的学习之旅！

# 学习资源

本页面汇总了学习过程中可能用到的各类资源。

## 资源分类

### 图片资源
- 课程配图
- 示意图和流程图
- 截图和演示图片

### 文件资源
- 示例代码
- 配置文件
- 参考文档

### 组件资源
- **Giscus 评论组件** - 可复用的评论系统组件
- 查看 [Giscus 配置指南](GISCUS_SETUP.md) 了解如何配置评论系统

## 使用说明

所有资源文件都存储在对应的子目录中，课程内容会根据需要引用这些资源。

### 引用图片

在 Markdown 文件中使用相对路径引用图片：

```markdown
![图片描述](../resources/images/example.png)
```

### 使用评论组件

Giscus 评论系统已集成到所有课程页面。如需在新页面添加评论功能，参考 `components/giscus-comments.html` 文件。
